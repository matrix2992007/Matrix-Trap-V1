# 🦅 Matrix Trap V1.0.48 - نظام التوعية الأمنية المتقدم

مرحباً بكم في مشروع **Matrix Trap**. هذا النظام مصمم خصيصاً لأغراض **التوعية الأمنية (Cyber Security Awareness)** والتعليم الفني للطلاب والمهتمين بمجال حماية البيانات.

## 🎯 الهدف من المشروع
الهدف هو إثبات مدى سهولة سحب "البصمة الرقمية" (Metadata) لأي جهاز بمجرد الضغط على رابط مجهول، وتنبيه المستخدمين لعدم الوثوق في الروابط حتى وإن كانت تبدو رسمية.

## ⚠️ إخلاء مسؤولية (Disclaimer)
> **هذا المشروع مخصص للأغراض التعليمية فقط.** المبرمج غير مسؤول عن أي استخدام خاطئ أو غير قانوني لهذه الأداة. استخدام هذه الأداة في التجسس أو إلحاق الضرر بالآخرين يعرضك للمساءلة القانونية. **كن هاكر أخلاقي دائماً.**

## 🚀 المميزات التقنية
- **Stealth Mode:** سحب البيانات في الخلفية دون التأثير على تجربة المستخدم.
- **Device Fingerprinting:** سحب الـ IP، المدينة، شركة الاتصالات، ونوع نظام التشغيل.
- **Battery Status:** مراقبة نسبة شحن البطارية وحالة الشحن لحظياً.
- **Hidden Admin Panel:** لوحة تحكم مدمجة محمية بكلمة سر لتغيير الإعدادات دون تعديل الكود.
- **Telegram Integration:** ربط مباشر مع بوت تليجرام لإرسال التقارير فوراً.

## 🛠️ كيف يعمل السيستم؟
1. قم برفع الملفات (`index.html`, `style.css`, `script.js`) إلى مخزن جيت هب الخاص بك.
2. قم بتفعيل **GitHub Pages** من إعدادات المخزن.
3. افتح الرابط الناتج، وانزل لأسفل الصفحة واضغط على **"Build: V1.0.48-Matrix"**.
4. أدخل كلمة السر الخاصة بك لفتح لوحة التحكم وربط البوت الخاص بك.

## 📱 البيانات التي يتم توعيتك بشأنها:
- عنوان الـ **IP** الخاص بك وموقعك التقريبي.
- معلومات كاملة عن **المتصفح والجهاز**.
- **حالة البطارية** (لمعرفة مدى قدرة المواقع على قراءة حساسات الجهاز).
- **التفاعلات اللحظية** (متى بدأت بلمس الشاشة أو التمرير).
